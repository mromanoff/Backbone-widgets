###Backbone widget app

####Build application with multiple widgets on single page.
* Every widget treated as a separate app with own build.
* All common libraries are bundled to core.js file

#### Build on
* NodeJS
* Faker.js (mockup data)
* Browserify
* Backbone.js
* Marionette.js
* Handlebars
* D3.js
* Sass
* Karma runner
* Jasmine Unit test
